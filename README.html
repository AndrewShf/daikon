Daikon invariant detector distribution

This is the distribution of the Daikon invariant detector, Daikon version 5.1.13, released December 2, 2014.

If you are working with a Daikon distribution downloaded from the Daikon website, then most everything is set up and ready to go. You should start out by reading the documentation which includes installation instructions.

However, if you are working with source cloned from the Google Code repository Daikon, then please review the file README.source.
