Anonymous CVS for the Daikon project

There are two modules (directories) available via anonymous CVS from pag.lcs.mit.edu: invariants (the bulk of the Daikon invariant detection system, which is written in Java) and dfej (the Daikon front end for Java, which is written in C++).

To obtain the invariants module, execute the following commands. They create an invariants subdirectory in the current directory. Use the password research. 

    cvs -d :pserver:anonymous@pag.lcs.mit.edu:/g4/projects/invariants/.CVS login
    cvs -d :pserver:anonymous@pag.lcs.mit.edu:/g4/projects/invariants/.CVS co invariants

Optionally, to obtain the dfej module, execute the following commands. They create a dfej subdirectory in the current directory. Use the password research. 

    cvs -d :pserver:anonymous@pag.lcs.mit.edu:/g4/projects/invariants/.CVS login
    cvs -d :pserver:anonymous@pag.lcs.mit.edu:/g4/projects/invariants/.CVS co dfej

After executing these commands, you should be able to issue CVS commands such as 

  cvs update
to update your copy of the repository from the master version.

Be sure to read over the "Installing Daikon" section of the Daikon User Manual for details on setting your classpath and the like.

Also, you may find the internal MIT instructions in directory invariants/doc/www/mit/ helpful in building Daikon. In particular, you may wish to ensure that you are using version 2 of Daikon.
