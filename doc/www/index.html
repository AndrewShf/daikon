The Daikon invariant detector

Daikon is a prototype implementation of dynamic detection of likely invariants.. Dynamic invariant detection runs a program, observes the values that the program computed, and then reports properties that were true over the observed executions. Daikon's output, which is called an "operational abstraction" looks like a formal specification, or the contents of assert statements. Examples include "a>0"; "c=2*d+3"; "array a is sorted"; "for all list objects lst, lst.next.prev = lst"; and the like. Daikon can detect properties in Java, Perl, C, and IOA programs; in spreadsheet files; and in other data sources. You can extend Daikon to add new properties or new languages.

Daikon is freely available, with both source code and documentation. Many researchers and practitioners have used Daikon, and Daikon itself is described in various publications.
