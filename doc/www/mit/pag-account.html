Creating an account on geyer

To create an account on geyer, perform the following two steps:

  1. Run inquir (for instance, on oregano, and using the .el rather than .elc file) to get a UID. Set the name and mail forwarding, at least, and note the UID for use in the useradd command below.

    If it is impossible to run inquir, then look at the end of /etc/passwd and choose a UID from there. Don't just omit the "-u UID" argument from the useradd command.

  2. Create a new account on geyer like so: 
    su
useradd -d /g5/users/username -u UID username
# Add to "invariants" and "pag" groups
emacs /etc/group
(cd /var/yp; make)
yppasswd username
# Add to "daikon" mailing list
emacs /etc/aliases
newaliases
cd ~username
echo "Adjust directory permissions"
chmod og+rx .
echo "Set up dotfiles, including email forwarding to Athena"
cp /g2/users/mernst/research/invariants/doc/www/mit/.cshrc .
echo "" >> .bashrc
cat /g2/users/mernst/research/invariants/doc/www/mit/.bashrc >> .bashrc
cp /g2/users/mernst/research/invariants/doc/www/mit/.cvsrc .
echo "" >> .emacs
echo "" >> .emacs
cat /g2/users/mernst/research/invariants/doc/www/mit/.emacs >> .emacs
echo username@mit.edu > .forward
chown username .cvsrc .emacs .forward
chgrp username .cvsrc .emacs .forward
exit
To remove an account, perhaps make a tar file of the user's directory, then do 
  userdel -r USERNAME
