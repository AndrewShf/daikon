Creating an account on geyer

To create an account on geyer, perform the following two steps:

Run inquir (on oregano, and using the .el rather than .elc file) to get a UID. Set the name and mail forwarding, at least.

Create a new account on geyer like so: 

su
useradd -d /g1/users/username -u UID username
passwd username
# Add to "invariants" group
emacs /etc/group
(cd /var/yp; make)
# Add to "daikon" mailing list
emacs /etc/aliases
exit
